精日校园腾讯云自动打卡方法

2022-01-22

详情在这个大佬的代码里github,使用的也是这个大佬的代码

腾讯云设置

点击这个去腾讯云注册一个账号，然后实名认证，之后点击这个去往云函数

在新建处新建函数

函数名称可以随便选，在运行环境这一处可以选择python3.7，但是前提要在代码设置好的情况下，在打包好代码上传，有点麻烦，或者你是大佬可以用这个。我建议选择python3.6。

执行时间设置60秒左右
完成

代码设置

点击函数代码进入控制台界面
点击终端-新终端

输入
建议一行行复制输入

cd src
rm -rf *
git clone https://github.com/CarltonHere/auto-cpdaily.git
cd auto-cpdaily
mv * ..
cd ..
rm -rf auto-cqdaily
pip3 install -r requirements.txt -t ./ -i https://mirrors.aliyun.com/pypi/simple

设置config.yml，下面这些要自己设置，yml里说的很清楚了，如果是我学校的直接点击这个复制我的表单设置

    - user:
        # 任务类型（收集：0、签到：1、查寝：2、政工签到：3、教师工作日志：4）
        type: 3
        # school 学校全称
        schoolName: 
        # username 学号或者工号
        username: ''
        # password 密码
        password: ''
        # 地址信息
        address: ''
        notifyOption: 
            # method和顶部参数一致，不填默认按照顶部参数执行
            # method: 0
            # rcvAcc为接受消息的账号，可以为Email账号、QQ号、或者是企业微信ID(@all为全体)
            rcvAcc: ''
        # 附上经纬度查询地址（请自行选择自己的学校地址，address，lon，lat都要填查询到的）：http://api.map.baidu.com/lbsapi/getpoint/index.html
        # lon 经度
        lon: 104.616858
        # lat 纬度
        lat: 28.793022
        # 是否检查表单的标题（1代表检查，0代表不检查）
        checkTitle: 1

贴士

  • 中间的第6个表单是要写自己的位置，注意填写。
  • 中间的ignore: True值为忽略函数，不会管这个标题，但是这个标题他必须存在，因为在填写时，手机上看到的默是填写好的，自动隐藏
